note right of Pipeline
  Verkoopfunnel met geordende fasen voor lead.
  Initiële records:
  - type lead; Hernia, Privatescan
  - type backoffice; Hernia, Privatescan
  Elke pipeline heeft zijn eigen fasen.
end note

note right of Lead
  Potentiële klant, optioneel gekoppeld aan een gebruiker en contactpersoon.
  Pipeline\-fasen ("nieuw", "in behandeling", "afgehandeld").
end note

note right of Order
  Afgesloten deal, gekoppeld aan een lead, gebruiker en contactpersoon.
end note

note right of Backoffice
 Automatiseringsflow gekoppeld aan een pipeline\-fase.
end note

note right of Activity
  Telefoongesprek \| Taak of geautomatiseerde stap in een Backoffice, optioneel toegewezen aan een gebruiker.
end note

note right of ContactPerson
  Externe contactpersoon betrokken bij leads of orders, geen gebruiker in het systeem.
end note

note right of Anamnesis
  Medische geschiedenis gerelateerd aan een lead, gekoppeld aan een gebruiker. (anamnese)
