note right of Pipeline
  Verkoopfunnel met geordende fasen voor lead.
  Initiële records:
  - type lead; Hernia, Privatescan
  - type backoffice; Hernia, Privatescan
  Elke pipeline heeft zijn eigen fasen.
end note

note right of Lead
  Potentiële klant, optioneel gekoppeld aan een gebruiker, contactpersoon en organisatie.
  Pipeline-fasen ("nieuw", "in behandeling", "afgehandeld").
  Ondersteunt nu persoonlijke velden en externe ID's.
  Contactpersoon is de primaire aanspreekpunt voor communicatie en overleg.
end note

note right of Order
  Afgesloten deal, gekoppeld aan een lead, gebruiker en contactpersoon.
  Ondersteunt nu factuur- en verzendadressen.
  Status wordt automatisch bijgewerkt op basis van OrderItem statussen:
  - INGEPLAND: alle items zijn klaar (INGEPLAND of NIEUW)
  - Anders: minimaal één item moet nog worden ingepland
end note

note right of OrderItem
  OrderItem status wordt automatisch bepaald:
  - INGEPLAND: als er een ResourceOrderItem bestaat voor deze regel
  - MOET_WORDEN_INGEPLAND: als het product partnerproducten heeft
  - NIEUW: standaard status
end note

note right of OrderCheck
  Checklist items voor een order.
  - removable: bepaalt of de check handmatig verwijderd kan worden
  - done: voltooiingsstatus van de check
  - Automatisch gegenereerde checks voor partner product rapportages zijn niet verwijderbaar
end note

note right of SalesLead
  Automatiseringsflow gekoppeld aan een pipeline-fase.
  Vervangt de oude "Backoffice" entiteit.
  Contactpersoon is de primaire aanspreekpunt voor communicatie en overleg.
end note

note right of Activity
  Telefoongesprek | Taak of geautomatiseerde stap in een SalesLead, optioneel toegewezen aan een gebruiker of groep.
  Kan nu direct gekoppeld zijn aan een lead en meerdere e-mails.
end note

note right of Person
  Externe contactpersoon betrokken bij leads of orders, kan gekoppeld zijn aan een gebruiker of organisatie.
  Ondersteunt nu uitgebreide persoonlijke informatie.
end note

note right of Organization
  Bedrijfsentiteit waaraan personen en leads gekoppeld kunnen zijn.
  Adres is verplicht bij aanmaken en wijzigen.
end note

note right of Address
  Adresinformatie die gekoppeld kan zijn aan leads, personen of organisaties.
  Ondersteunt nu meerdere entiteitstypen.
end note

note right of Department
  Afdelingen voor het organiseren van gebruikers en leads.
  Hernia en Privatescan zijn de belangrijkste afdelingen.
end note

note right of Anamnesis
  Medische geschiedenis gerelateerd aan een lead of persoon, gekoppeld aan een gebruiker. (anamnese)
  Uitgebreide medische velden toegevoegd.
end note

note right of Email
  E-mail communicatie gekoppeld aan leads, personen en activiteiten.
  Ondersteunt e-mail threads via parent_id relatie.
  Kan gekoppeld zijn aan nul of één activiteit (0..1).
end note

note right of ResourceAvailability
    Weekday: 0=Sunday, 1=Monday, ...
end note

note right of ImportRun
  SugarCRM import run logging voor leads, persons, users en email-attachments.
  Status: running, completed, failed.
end note

note right of ImportLog
  Individuele log entries (error, warning, info) voor een ImportRun.
  Context bevat extra informatie zoals record_id en error details.
end note
